brew 降级

February 20, 2025

参考: Homebrew install specific version of formula?

以alt-tab为例，从7.20.0 (有bug会崩溃) 降级到7.19.1

  1. 打开 https://formulae.brew.sh/cask/alt-tab
  2. 跳转 Cask code on GitHub
  3. 点开 alt-tab.rb History
  4. 找到要目标版本 (要降低到的版本)，点View code at this point
  5. 下载 alt-tab.rb 7.19.1
  6. 控制台进alt-tab.rb所在目录，执行brew rnstall alt-tab.rb

该方法虽然麻烦，但通用。如果软件预备了多版本，例如 mysql-client，可以通过brew install mysql-client@8.4切版本，相对方便
